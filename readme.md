# GAME班における研究活動のすすめ

## 本資料の概要
GAME班とは，田中輝雄教授の先進ソフトウェア研究室，および藤井昭宏准教授の高性能計算研究室における研究グループのひとつであり，２０２４年度（令和６年度）から始動した．

本資料では同年度のGAME班の研究活動から，今後活用できそうな情報について書く．
